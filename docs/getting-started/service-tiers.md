# Service Tier Comparison

F5 Distributed Cloud Services offers **Standard** and **Advanced** tiers across its service portfolio. Understanding these differences is critical for selecting the appropriate tier for your requirements.

## Tier Overview

| Aspect | Standard | Advanced |
|--------|----------|----------|
| **Target Use Case** | Basic security and networking | Complex, high-security environments |
| **API Security** | Schema-based protection | Behavioral analysis + ML-based detection |
| **Bot Defense** | Signature-based | Behavioral + AI/ML-powered |
| **DDoS Protection** | L3/L4 mitigation | L3/L4 + L7 application-layer mitigation |
| **Managed Services** | Basic support | 24x7 SOC + dedicated resources |

## Licensing Models

### Annual Enterprise Subscription

- Fixed annual commitment
- Predictable costs
- Includes base entitlements
- Add-ons available for expansion

### Pay-As-You-Go (AWS Marketplace)

- No upfront commitment
- Monthly billing in arrears
- Ideal for variable workloads
- Base package starts at ~$3.70/hour

### Flexible Consumption Program (FCP)

- 3-year minimum commitment
- Annual spending flexibility
- Best pricing tier
- True-up/true-forward adjustments

## Which Tier Do You Need?

### Choose Standard If:

- [x] You have straightforward security requirements
- [x] Your applications don't require advanced bot protection
- [x] L3/L4 DDoS protection is sufficient
- [x] You have in-house security operations capability
- [x] Cost optimization is a primary concern

### Choose Advanced If:

- [x] You face sophisticated bot attacks
- [x] You need L7 DDoS protection
- [x] Regulatory compliance requires advanced controls
- [x] You have extensive API exposure
- [x] You need 24x7 SOC support
- [x] You're a high-value target for attackers
